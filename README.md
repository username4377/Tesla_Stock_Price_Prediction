## 📖 About This Application

Tesla_Stock_Price_Prediction is a desktop application designed to help you forecast Tesla's closing stock prices. It uses historical stock data and a smart machine learning model called XGBoost to find patterns in the price changes over time. This lets you see potential future prices based on past trends.

You do not need any special knowledge to use this app. The program handles all the technical details in the background. This tool can support financial enthusiasts, investors, or anyone curious about Tesla stock trends.

## 🔍 How It Works (In Simple Terms)

Tesla_Stock_Price_Prediction uses a machine learning technique called XGBoost. It looks at past Tesla stock prices over time using a sliding window method — this just means it examines groups of days to find trends. The app learns these patterns and uses them to guess future prices.

You don’t need to understand the details to use the app. Just know it applies advanced math based on historical stock data to help you see possible price movements.
